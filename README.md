# 🎮 BUILD — 8-Bit Portfolio Game

**Your resume, delivered through an 8-bit game.** Collect tools, dodge spikes, reach the door — then a terminal reveals your portfolio. Zero dependencies, single HTML file. Host on GitHub Pages.

Hiring managers click a link, play for 60 seconds, and *remember you.*

## How to use
1. Edit `game.html` — change `COMPANY` and `PORTFOLIO` at the top
2. Enable GitHub Pages on this repo → Settings → Pages → deploy from `main`
3. Send the link: `https://realMNohgee.github.io/portalfolio`
4. They play. They see your work. You get the interview.

## Customize per company
```js
const COMPANY = "Structured AI";       // changes instantly
const ACCENT  = "#ff8800";             // orange for construction
const PORTFOLIO = `...your bio...`;    // full portfolio text
```

## Controls
Arrow keys / WASD to move. Space to jump. Collect all tools. Reach the door.
